# В результате выполнения ДЗ студент подготовит стенд на Vagrant
> Две машины Клиент и сервер

> На клиенте устанавливается ansible 

> Описываем host.ini file

> Создаем структура папок: mysql, nginx, php

> В папке tasks описываем main.yml file

> Роли описываем в playbook.yml 

> С помощбю команды ansible-playbook playbook.yml -i hosts.ini запускаем.
